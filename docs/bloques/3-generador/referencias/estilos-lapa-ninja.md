# Referencia — Despensa de estilos (lapa.ninja)

## Qué es
Pool de **géneros de layout/composición** (no el ADN de una web concreta) sacado de la taxonomía de categorías de [lapa.ninja](https://www.lapa.ninja/) (galería de showcases). Complementa:
- `despensa.md` → ADN extraído 1:1 de una web concreta vía `skillui`.
- `tipografias.md` → pares tipográficos.

Aquí se elige **género de composición + paleta-base**, no una web de referencia.

## Cómo se usa
En Etapa 3b, impeccable elige **un género distinto de este pool por cada preview del grupo "lapa.ninja"**, evitando repetir el arquetipo de despensa usado en el grupo "propias". Objetivo: que las 9 previews cubran lenguajes visuales realmente distintos entre sí, no variaciones del mismo.

## Categorías curadas (perfil EVOLink → género lapa.ninja → notas)
| Perfil EVOLink | Género lapa.ninja | Composición típica | Encaja en |
|---|---|---|---|
| corporativo-confianza | **Corporate / Business** | Grid ordenado, secciones con bordes finos, datos en tarjetas/stats | mudanzas, reformas, asesorías |
| servicios-local premium | **Bento Grid** | Bento de tamaños desiguales, cards icono+texto, ritmo visual variado | peluquerías, talleres, clínicas |
| editorial | **Magazine / Editorial** | Tipografía grande, columnas asimétricas, mucho texto cuidado, poca decoración | artesanos, estudios, storytelling |
| minimal-lujo | **Minimal / Studio** | Espaciado generoso, monocromo + 1 acento, pocas formas, mucho aire | inmobiliarias, interiorismo |
| kinético / personalidad fuerte | **Retro Style / Pattern** | Bloques de color sólido, formas geométricas repetidas, bordes gruesos | marca con carácter, talleres creativos |
| marketplace / catálogo | **Real Estate / Furniture & Interiors** | Cards de producto/servicio con imagen protagonista, precios/specs visibles | tiendas, mobiliario, inmobiliaria |
| tech-friendly local | **SaaS / Productivity** suavizado | Hero con producto/servicio "como app", iconografía limpia, secciones cortas | gestorías, clínicas con cita online |

## Reglas (igual que el resto del flujo)
- **No clonar** HTML, copy ni imágenes de ningún ejemplo de lapa.ninja: solo el "idioma" de composición/paleta/tipografía — mismo principio que `flujo-previews.md` aplica a la preview de referencia única.
- Honestidad, placeholders de logo/foto y `.reveal` siguen las reglas globales del flujo.
- Evitar que dos previews del mismo lote (propias / lapa.ninja / stitch) usen el mismo género o el mismo par tipográfico.

## Cómo ampliar
Visitar `lapa.ninja/categories/<categoria>`, mirar 3-5 ejemplos, anotar en una fila nueva qué patrón de composición/paleta lo define. Curación humana de los socios — no automatizar el scraping de lapa.ninja (es una galería con derechos de terceros; solo inspiración de lenguaje, igual que cualquier referencia visual).
