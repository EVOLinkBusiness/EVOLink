# Referencia — Despensa de estilos (lapa.ninja)

## Qué es
Pool de **géneros de layout/composición** (no el ADN de una web concreta) sacado de la taxonomía de categorías de [lapa.ninja](https://www.lapa.ninja/) (galería de showcases). Complementa:
- `despensa.md` → ADN extraído 1:1 de una web concreta vía `skillui`.
- `tipografias.md` → pares tipográficos.

Aquí se elige **género de composición + paleta-base**, no una web de referencia.

## Cómo se usa
En Etapa 3b, impeccable elige **un género distinto de este pool por cada preview del grupo "lapa.ninja" (previews 4-6)**, evitando repetir el arquetipo de despensa usado en el grupo "propias" (1-3) y el lenguaje del grupo GSAP (7-12). Objetivo: que las 12 previews cubran lenguajes visuales realmente distintos entre sí, no variaciones del mismo.

## Categorías curadas (perfil EVOLink → género lapa.ninja → notas)

### Por tipo de negocio / layout
| Perfil EVOLink | Género lapa.ninja | Composición típica | Encaja en |
|---|---|---|---|
| corporativo-confianza | **Corporate / Business** (~2.450 ejs.) | Grid ordenado, secciones con bordes finos, stats/testimonios en tarjetas, CTA prominente | mudanzas, reformas, fontaneros, asesorías |
| agencia / estudio creativo | **Agency** (~960 ejs.) | Hero full-bleed con tipografía bold, scroll-personality, portfolio grid o case studies | peluquerías premium, estudios diseño, interiorismo |
| servicios-local premium | **Bento Grid** | Bento de tamaños desiguales, cards icono+texto, ritmo visual variado | peluquerías, talleres, clínicas |
| editorial | **Magazine / Editorial** | Tipografía grande, columnas asimétricas, mucho texto cuidado, poca decoración | artesanos, estudios, storytelling |
| minimal-lujo | **Minimal / Studio** | Espaciado generoso, monocromo + 1 acento, pocas formas, mucho aire | inmobiliarias, interiorismo |
| kinético / personalidad fuerte | **Retro Style / Pattern** | Bloques de color sólido, formas geométricas repetidas, bordes gruesos | marca con carácter, talleres creativos |
| marketplace / catálogo | **Real Estate / Furniture & Interiors** | Cards de producto/servicio con imagen protagonista, precios/specs visibles | tiendas, mobiliario, inmobiliaria |
| tech-friendly local | **SaaS / Productivity** suavizado | Hero con producto/servicio "como app", iconografía limpia, secciones cortas | gestorías, clínicas con cita online |

### Por paleta de color
| Paleta | Señal lapa.ninja | Efecto visual | Cuándo usar |
|---|---|---|---|
| **Black / Oscuro** | categoría "Dark" | Impacto máximo, aire premium-nocturno | restaurantes, bares, marcas de lujo, fitness |
| **White / Claro** | categoría "Light" / "Minimal" | Limpieza, confianza, espacio de respiración | clínicas, reformas, floristería, joyería |
| **Blue** | categoría "Blue" | Profesionalidad, fiabilidad, tecnología suave | asesorías, inmobiliarias, transporte |
| **Gradient / Colorido** | categoría "Gradient" | Energía, modernidad, dinamismo | fitness, wellness, educación, apps locales |

### Por plataforma de construcción (referencia de calidad de layout)
| Plataforma | Señal lapa.ninja | Por qué vale como referencia |
|---|---|---|
| **Webflow** | categoría "Webflow" | Layouts pulidos con scroll-driven sin código; buena señal de ritmo editorial y tipografía |
| **Framer** | categoría "Framer" | Webs con motion y microinteracciones por defecto; muy útil como referencia para previews GSAP |

## Reglas (igual que el resto del flujo)
- **No clonar** HTML, copy ni imágenes de ningún ejemplo de lapa.ninja: solo el "idioma" de composición/paleta/tipografía — mismo principio que `flujo-previews.md` aplica a la preview de referencia única.
- Honestidad, placeholders de logo/foto y `.reveal` siguen las reglas globales del flujo.
- Evitar que dos previews del mismo lote (propias / lapa.ninja / GSAP) usen el mismo género o el mismo par tipográfico.

## Cómo ampliar
Visitar lapa.ninja y filtrar por categoría, mirar 3-5 ejemplos, anotar en una fila nueva qué patrón de composición/paleta lo define. Curación humana de los socios — no automatizar el scraping de lapa.ninja (es una galería con derechos de terceros; solo inspiración de lenguaje, igual que cualquier referencia visual).
