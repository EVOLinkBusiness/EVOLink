# ROADMAP.md — Webs Javi

Fases de alto nivel. Se refina en cada brainstorming. La fase activa se marca aquí y en `HANDOVER.md`.

- **F0 — Infraestructura** ✅ *(completada esta sesión)*
  Repo git, metodología (CLAUDE/HANDOVER/docs), comandos `/inicio` `/cierre`, hooks, memoria.
- **F1 — Brainstorming y descomposición** ⏳ *(siguiente)*
  Descomponer el negocio en subsistemas; diseñar el primero; primera spec en `docs/superpowers/specs/`.
- **F2 — Lead-gen MVP**
  Google Places API → pipeline que produce una lista cualificada de negocios sin web.
- **F3 — Pipeline de producción web**
  De brief/ficha a web a medida con skills IA; plantilla de proyecto, sistema de diseño, QA.
- **F4 — Ventas / outreach**
  Contacto automatizado con demo/preview previa; seguimiento; cierre.
- **F5 — Entrega / hosting / billing**
  Deploy automatizado + dominio + Stripe (setup + suscripción) + onboarding del cliente.
- **F6 — Marketing automation**
  Captación y nurturing continuos a largo plazo.

> El orden de F2–F6 puede ajustarse tras el brainstorming (p. ej. validar producción antes que captación). Cada fase pasa por su ciclo superpowers: spec → plan → código.
