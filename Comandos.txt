[ AÑADIR REFERENCIA NUEVA ]
----------------------------

# Paso 1 · Extraer ADN visual (motion + feel)
npx skillui@latest --url <URL> --mode ultra --out .claude/skills/referencias-visuales/<nombre>

# Paso 2 · Extraer tokens estructurados (color / espaciado / radio)
npx extract-design-system <URL>
cp -r .extract-design-system/ .claude/skills/referencias-visuales/<nombre>/tokens/

# Paso 3 · Limpiar restos si aparecen
rm -rf .agents/ skills-lock.json .extract-design-system/

# Paso 4 · Generar la ficha (prompt para Claude Code)
# Pega esto en Claude Code tras el paso 2:
#
#   Lee los archivos de .claude/skills/referencias-visuales/<nombre>/
#   y .claude/skills/referencias-visuales/<nombre>/tokens/
#   y genera la fila exacta para añadir a la tabla "Inventario admitido"
#   de docs/bloques/3-generador/referencias/indice-referencias.md.
#   Formato:
#   | nombre | fuente | perfil | roles | mood | motion (1-5) | estructura/solo-idioma | OK | ruta-tokens |
#   Usa solo los perfiles y roles válidos del índice.
#   Justifica en una línea el motion y el tipo elegido.
#
# Claude Code te devuelve la fila lista. La revisas y la pegas
# en docs/bloques/3-generador/referencias/indice-referencias.md
# bajo la tabla "Inventario admitido".

# Paso 5 · Commitear y subir
git add .claude/skills/referencias-visuales/<nombre>/ \
        docs/bloques/3-generador/referencias/indice-referencias.md
git commit -m "feat: referencia <nombre> al indice (perfil: <perfil>)"
git push
